# Plan: Backend API for Strategy Results

Expose simulation results and daily performance data via FastAPI endpoints to power the React dashboard.

## Context
Phase 3 requires a bridge between the SQLite database and the React frontend. We need endpoints that allow the frontend to:
1. List all simulations (to build selectors).
2. Fetch a specific simulation's details (for metrics summary).
3. Fetch daily performance data (for charting).
