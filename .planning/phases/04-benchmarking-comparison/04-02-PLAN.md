# Plan 04-02: Frontend Benchmark Overlay & Stats

## Objective

This plan brings the backend benchmarking work into the UI. We will update the `StrategyChart` to allow overlaying a benchmark's performance on top of the selected strategy's performance, and update the `StatsGrid` to display the newly refined Sharpe and Sortino ratios.

## Context

The backend now identifies benchmarks using the `is_benchmark` flag. The frontend needs to fetch this data and provide a way for users to select an overlay. We use `lightweight-charts`, which supports multiple line series on a single chart.
